# Amazon Product Analytics Dashboard

## Project Overview

Developed an end-to-end Product Analytics Dashboard to analyze Amazon product data and uncover insights related to ratings, reviews, discounts, and pricing trends. The project involved data cleaning and preprocessing using Python, followed by interactive dashboard development in Tableau.

---

## Tools & Technologies

- Python (Pandas, NumPy)
- Tableau
- Microsoft Excel
- CSV Dataset

---

## Data Preparation

- Cleaned and transformed raw Amazon product data using Python.
- Handled missing values and inconsistent records.
- Converted data types for numerical analysis.
- Created derived fields such as Main Category, Rating Category, and Review Length.
- Exported a cleaned dataset for visualization in Tableau.

---

## Dashboard Features

### KPI Cards
- Total Products
- Average Rating
- Average Discount Percentage
- Total Reviews

### Visualizations
- Average Rating by Category
- Top Reviewed Products
- Top Rated Products
- Price vs Rating Analysis (Scatter Plot)

---

## Key Insights

- Office Products recorded the highest average ratings.
- Electronics and Computer Accessories generated the highest customer engagement.
- Several highly-rated products had relatively low review counts.
- Product ratings remained consistently high across different price ranges.
